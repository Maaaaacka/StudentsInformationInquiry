# 学生信息查询系统

## 项目简介

这是一个采用苹果简约风格设计的学生信息查询系统，支持通过姓名、学号、证件号码等关键词进行模糊搜索，并以卡片形式展示学生信息。

<img width="879" height="508" alt="页面" src="https://github.com/user-attachments/assets/207d5685-0b8e-4748-ba67-aa4b48bd91a1" />


## 功能特点

1. **搜索功能**：支持通过姓名、学号、证件号码等关键词进行模糊搜索
2. **卡片展示**：搜索结果以卡片形式展示，包含学生基本信息和头像
3. **展开功能**：点击卡片可展开查看完整个人信息，包括年龄计算
4. **响应式设计**：适配不同屏幕尺寸，提供良好的用户体验
5. **本地图片支持**：使用本地img文件夹中的图片作为学生头像

## 文件结构

```
student-search-system/
├── index.html          # 主页面
├── img/                # 图片文件夹
│   ├── 姓名_证件号码.jpg  # 学生头像图片
│   └── default_avatar.jpg # 默认头像
└── README.md           # 项目说明
```

## 图片使用说明

1. 将学生头像图片放置在`img`文件夹中
2. 图片文件命名格式必须为：`姓名_证件号码.jpg`
   - 例如：`黄存正_330302200710242437.jpg`
3. 如果图片文件不存在或加载失败，系统会自动显示默认头像`default_avatar.jpg`

## 如何使用

1. 克隆或下载本项目到本地
2. 将学生头像图片按照指定格式命名并放入`img`文件夹
3. 直接打开`index.html`文件即可使用系统
4. 在搜索框中输入关键词（姓名、学号、证件号码等）进行搜索
5. 点击搜索结果卡片可展开查看详细信息

## 数据说明

系统中的学生数据已预设在JavaScript代码中，包含以下字段：
- 院校名称
- 姓名
- 证件号码
- 出生日期
- 性别
- 民族
- 所学专业
- 班级
- 学号
- 学制
- 入学时间
- 学籍编号

如需修改或添加学生数据，请直接编辑`index.html`文件中的`students`数组。

## 注意事项

1. 确保图片文件路径和命名格式正确，否则将显示默认头像
2. 建议使用高清头像图片，以获得最佳显示效果
3. 系统支持模糊搜索，输入部分关键词即可
4. 点击卡片头部区域可展开/折叠详细信息
